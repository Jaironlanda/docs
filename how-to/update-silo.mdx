## Introduction

This page provides a comprehensive guide on how to update silo data through the user console.

## JSONsilo Editor

The JSONsilo editor is a crucial tool that allows you to manage JSON content more easily. It provides an intuitive interface for editing and updating your silo data.

<img className="rounded" height="200" src="/images/update-silo/base-editor-5.png" />

<img className="rounded" height="200" src="/images/update-silo/base-editor-4.png" />

1. Basic information about the silo: Name, UUID, and region.
2. Available features in the editor:
  - **Manual**: Manual input
  - **Upload**: Upload `.json`
  - **Download**: Download silo to `.json`
  - **Save**: Save JSON file to the server and redeploy the silo
  - **Save & Close editor**: Save JSON file and close the editor
3. Editor, where you can edit and update JSON content
4. Exit button, which will redirect you to the view silo page
5. Toggle button to change the editor color: Light mode and Dark mode
